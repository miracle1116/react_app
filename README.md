1. A simple web app that developed to provide a **reservation platform** for reserve a time slot of discussion room in Kolej Kediaman Kinabalu. <br />
2. This was developed using **html**, **css** and **javascript** with **react** and hosted on **firebase**. <br />
3. The database used was **firebase firestore** that provide a **real time database** and fulfil the feature that the web needed. <br />
